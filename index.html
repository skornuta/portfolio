Spotify Album Art Display

In Progress

ESP32-D device that fetches the current Spotify track over Wi-Fi and renders the album cover on a 240×240 round GC9A01 SPI display. Handles OAuth2 token refresh automatically, decodes JPEG frames on-device using TJpg_Decoder with LovyanGFX as the display driver, and only redraws when the track or album image changes. Handles 204 (nothing playing) and 401 (token expired) responses. Currently working through the firmware — physical build and enclosure are the next phase.

In Progress

Flipper Zero-inspired handheld device. 16×2 I²C LCD (0x27) with a 4-button debounced menu on GPIO 25/26/27/33. CC1101 Sub-GHz transceiver over VSPI scans 315–915 MHz with live RSSI readout. IR receiver on GPIO 34 via IRremoteESP8266. Each hardware module compiles only when its library is present.

WS2812B LED Strip Controller

8-LED WS2812B strip driven from GPIO2 using FastLED in C. Covers GRB channel ordering, FastLED.setBrightness() for PWM control, and PlatformIO project structure. A good starter build for understanding addressable LED protocols before adding more complexity.

About

I'm a high school student in San Jose building embedded systems projects. Most of what I work on is hardware-first: pick a microcontroller, wire up the components, write the firmware, and figure out the software side as the project demands it.

My usual setup is ESP32 with PlatformIO and the Arduino framework in C/C++, running Windows with Zed or VS Code. When a project needs a physical enclosure, I design it in FreeCAD.

Looking for robotics clubs, engineering internships, and open-source hardware projects to contribute to.
